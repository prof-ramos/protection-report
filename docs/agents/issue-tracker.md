# Issue Tracker — GitHub Issues

This repository tracks issues using GitHub Issues via the `gh` CLI (`prof-ramos/protection-report`).

## Workflow

- **Read issues**: `gh issue list` / `gh issue view <id>`
- **Create issues**: `gh issue create --title "<title>" --body "<body>"`
- **Close issues**: `gh issue close <id>`

## External PRs
- **PRs as a request surface**: `false`
