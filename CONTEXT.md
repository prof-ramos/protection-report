# 📖 Protection Report — Context & Domain Glossary (`CONTEXT.md`)

## Ubiquitous Language & Core Terminology

- **Account**: A public user profile found on a platform (e.g. YouTube, Instagram, GitHub) during an OSINT scan.
- **Source / Parser**: An OSINT tool JSON output format (e.g. `maigret`, `sherlock`, `blackbird`, `naminter`, `enola`, `vesper`).
- **Cluster**: A grouped identity formed by matching accounts sharing normalized full names, handles, or emails.
- **Risk Score**: A 0-10 weighted rating assessing digital exposure and privacy risk.
- **Score Breakdown**: Evidence-backed breakdown of point contributions per risk rule.
- **Redaction / PII Masking**: Privacy option (`--redact`) to obscure sensitive fields (e.g. `g***@domain.com`).
