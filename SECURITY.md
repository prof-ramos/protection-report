# Policy de Segurança / Security Policy

## Versões Suportadas / Supported Versions

| Versão / Version | Suportada / Supported |
| ---------------- | --------------------- |
| 1.0.x            | :white_check_mark:    |
| < 1.0.0          | :x:                   |

## Reportando uma Vulnerabilidade / Reporting a Vulnerability

Se você encontrar uma vulnerabilidade de segurança no `protection-report`, por favor informe com responsabilidade:

1. **Não** abra uma issue pública referente a uma vulnerabilidade ainda não corrigida.
2. Envie os detalhes técnicos da vulnerabilidade para os mantenedores do repositório.
3. Inclua os passos necessários para reproduzir o problema, impactos potenciais e qualquer código de demonstração (PoC).

Analisaremos o relatório em até 48 horas e forneceremos uma previsão para mitigação e lançamento de um patch de correção.
